# Mondegreen-REACT

## Install Dependencies
From the root, run

```
yarn

```
## Run 
*Make sure Monogdb is running in a sepreate terminal*

#### Dev Mode

```
nodemon yarn start
```

#### Prod Mode

```
nodemon yarn start-prod
```

## Redux overlay controls
#### Hide Toggle
Control + H

#### Change position in window
Control + Q
